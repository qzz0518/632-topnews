# NEWS / TERMINAL

一个以工业设备面板为设计风格的全球新闻聚合器，支持来自中国、日本、美国、英国等多个国家和地区的新闻源。

## 功能特点

- **工业面板UI**: 采用 System Baustein 风格设计，CRT显示屏、硬件按钮、单间距字体
- **全球新闻源**: 集成中国(头条)、日本(NHK)、美国(CNN)、英国(BBC)等新闻源
- **区域切换**: 通过硬件风格按钮过滤不同国家/地区的新闻
- **智能分类**: 根据关键词自动分类（综合、科技、财经、体育、娱乐）
- **新闻收藏**: 一键收藏感兴趣的新闻，保存至浏览器本地存储
- **CRT新闻滚动**: 顶部模拟CRT屏幕实时滚动展示最新标题
- **响应式设计**: 适配桌面和移动设备
- **自动刷新**: 每30秒自动获取最新新闻

## 技术栈

- **Vite** — 开发服务器与构建工具
- **Vanilla JavaScript** — 无框架依赖
- **pnpm** — 包管理器
- **Inter + JetBrains Mono** — 字体
- **Vite Proxy** — 开发环境CORS代理

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```
