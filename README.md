# 热门新闻应用

一个简洁优雅的中文热门新闻Web应用，集成今日头条热榜数据，提供分类浏览和收藏功能，同时展示当前位置的天气信息。

## 功能特点

- **今日头条热榜**: 实时获取今日头条热门新闻，掌握最新资讯
- **智能分类**: 根据新闻标题关键词自动分类（综合、科技、财经、娱乐、体育）
- **新闻收藏**: 一键收藏感兴趣的新闻，方便后续查看
- **本地存储**: 收藏的新闻保存在浏览器本地存储中，关闭后不会丢失
- **实时天气**: 顶部显示当前位置的天气信息（温度、天气状况）
- **响应式设计**: 完美适配不同设备屏幕，包括手机、平板和桌面电脑
- **小红书风格UI**: 采用现代化的小红书风格设计，清新简约

## 技术栈

## 详细使用说明

### 浏览新闻

- 打开应用后，默认显示"综合"分类下的所有新闻
- 点击顶部的分类按钮可以切换不同的新闻分类
- 新闻以卡片形式显示，包含标题、图片、热度和发布时间
- 点击新闻卡片可在新标签页中打开原新闻页面

### 收藏功能

- 点击新闻卡片右上角的星形图标可收藏/取消收藏该新闻
- 点击顶部导航栏中的"收藏"按钮可查看所有已收藏的新闻
- 收藏的新闻会保存在浏览器的本地存储中，关闭页面后不会丢失
- 在"收藏"页面取消收藏后，该新闻会立即从列表中移除

### 天气功能

- 首次访问时，应用会请求获取您的地理位置以显示当地天气
- 如果您允许位置访问，顶部会显示当前位置的天气信息
- 如果您拒绝位置访问，应用会使用默认位置（北京）的天气数据

## 数据来源

- 新闻数据来自今日头条热榜
- 天气数据可通过OpenWeatherMap API获取（当前使用模拟数据）
